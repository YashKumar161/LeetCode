# Remove Duplicates from Sorted Array

## Problem Name
Remove Duplicates from Sorted Array

## Approach
Removing Duplicates from a Sorted Array (removeDuplicates method):
-The removeDuplicates method removes duplicate elements from the sorted array nums.
-It uses a pointer j to keep track of the next position for unique elements.

## Solution
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        j = 1
        for i in range(1, len(nums)):
            if nums[i] != nums[i - 1]:
                nums[j] = nums[i]
                j += 1
        return j
